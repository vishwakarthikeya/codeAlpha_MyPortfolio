# My Portfolio Website

This repository contains my personal portfolio website, developed using HTML, CSS, and JavaScript. It is designed to present my profile, highlight my projects, and provide easy access to my contact information.

---

## Overview

The portfolio serves as a central place to showcase my work, technical skills, and background. It follows a simple and structured design approach to ensure clarity and ease of navigation.

---

## Features

- Structured sections for profile, projects, and contact information  
- Clean and responsive layout  
- Use of images to represent personal and project-related content  
- Navigation using anchor links for a smooth browsing experience  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## Getting Started

To run this project locally:

1. Clone the repository:
   ```
   git clone  https://github.com/vishwakarthikeya/codeAlpha_MyPortfolio.git
   ```

2. Open the project folder.

3. Launch `index.html` in your browser.
